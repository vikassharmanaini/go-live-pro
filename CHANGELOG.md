# Changelog

All notable changes to the "Go Live Pro" extension will be documented in this file.

## [1.0.0] - Initial Release 🚀

- **Extension Launch**: Professional-grade Live Server for VS Code.
- **Embedded Preview**: Real Chromium browser inside VS Code using CDP.
- **Advanced DevTools**: Integrated Console, Elements Inspector, and Storage viewer.
- **Responsive View**: Presets for Mobile, Tablet, and Desktop.
- **Multi-root Support**: Run multiple servers simultaneously.
- **Hot Reload**: WebSocket-based instant updates.
- **SPA & Proxy support**: Perfect for React, Vue, and Angular developers.
- **Premium Settings**: Interactive Webview-based configuration.
